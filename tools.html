VinylDNS Tools

There are a few existing tools for working with the VinylDNS API.

  • vinyldns-cli - a VinylDNS API command line client
  • vinyldns-ruby - a VinylDNS Ruby gem
  • go-vinyldns - Go client package for VinylDNS
  • vinyldns-java - Java client for VinylDNS
  • vinyldns-python - Python client library for VinylDNS
  • vinyldns-js - JavaScript client for use in Node.js
  • terraform-provider-vinyldns - a Terraform provider for VinylDNS

Integrations

  • external-dns - DNS provider-agnostic syncronization of Cloud Foundry and Kubernetes resources, including VinylDNS
